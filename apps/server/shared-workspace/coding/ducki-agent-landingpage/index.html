DucKI Agent
GitHub Dokumentation AI Agent

Ich plane, codiere und liefere Features end-to-end.

DucKI ist ein self-hosted AI Agent fuer produktive Arbeit: mit Planung, Recherche, Workflows, persistentem Memory und sicherer Automatisierung im Projektkontext.

Faehigkeiten Benefits Quickstart

  • Recherche, Analyse und strukturierte Ergebnisaufbereitung
  • Workflow-Orchestrierung fuer mehrstufige Automationen
  • Memory-gestuetzte Kontextarbeit ueber Sessions hinweg
  • Sichere Tool-Ausfuehrung mit nachvollziehbaren Ergebnissen

Quickstart in 3 Minuten

Setup ist leichtgewichtig: installieren, .env anlegen, starten. 

pnpm install
Copy-Item .env.example .env
pnpm dev
  • Web UI: http://localhost:3000
  • API: http://localhost:3001
  • Health: http://localhost:3001/health

Setup Hinweise

  • In /settings zuerst Provider und Modell setzen.
  • In /skills nur benoetigte Skills aktivieren.
  • Mit /chat iterativ arbeiten, /workflow fuer Graph-Runs.
  • Aktive Agent-Laeufe ueber /agents beobachten.
