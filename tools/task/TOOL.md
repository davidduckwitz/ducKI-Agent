## Zweck
Task-Lebenszyklus (create/get/update/start/complete/fail/list) innerhalb
eines Projekts. Grundlage für Fortschritts-Tracking über mehrere Agent-Läufe
hinweg.

## Core-Tool
Dieses Tool ist als Core markiert und kann in den Settings nicht deaktiviert
werden — ohne Task-Tracking verliert der Agent seinen Arbeitsfortschritt.
