## Zweck
Ausgehende Nachrichten über konfigurierte externe Portale (Discord/Telegram/
Webhook).

## Optionales Tool
Standardmäßig deaktiviert. In den Settings unter "Tools" aktivieren, wenn der
Agent proaktiv über externe Kanäle senden soll.
