## Zweck
Zeitgesteuerte, wiederkehrende Ausführung von Tasks/Prompts/Tools/Skills.

## Optionales Tool
Standardmäßig deaktiviert. In den Settings unter "Tools" aktivieren, wenn der
Agent eigene Cronjobs anlegen/verwalten soll.
