## Zweck
Projekt-Verwaltung: Container für Tasks und Konversationen. Nutze `list` um
vorhandene Projekte zu sehen, bevor ein neues erstellt wird (Duplikate
vermeiden).

## Core-Tool
Dieses Tool ist als Core markiert und kann in den Settings nicht deaktiviert
werden — der Agent braucht es, um seinen eigenen Arbeitszustand zu verwalten.
