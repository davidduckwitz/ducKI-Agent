## Zweck
Verwaltung und Ausführung von Workflow-Graphen (mehrstufige, ggf.
wiederaufnehmbare Agent-Pipelines).

## Optionales Tool
Standardmäßig deaktiviert. In den Settings unter "Tools" aktivieren, wenn der
Agent Workflow-Graphen orchestrieren soll.
